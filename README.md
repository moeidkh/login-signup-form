# Login-singup form
* This is a React SPA (Single Page Application) project that uses the sweetalert2 library for displaying pop-up alerts. The project is also fully responsive, meaning it can be viewed on desktops, tablets, and mobile devices with ease.

## Installation
* The project is hosted online, so there's no need to install anything to use it. Simply navigate to https://preeminent-malasada-964f7b.netlify.app/ to view the application.

## Usage
* The application is a simple form that allows users to input their name, email, and a message. When the user clicks the "Submit" button, the form data is validated and a success or error message is displayed using sweetalert2.

* The application is fully responsive, meaning it can be viewed on any device with ease. The layout adjusts automatically to fit the screen size, providing an optimal user experience on all devices.
